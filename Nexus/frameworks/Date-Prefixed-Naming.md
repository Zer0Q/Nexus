# Date-Prefixed Naming

## Definition
File naming convention using YYYY-MM-DD-TYPE-TOPIC format. Date prefix auto-sorts chronologically, type identifier indicates content, topic identifier describes subject. Prevents naming conflicts and enables date-based retrieval.

## Why It Matters
Consistent naming means you can find any note by typing a partial match into search and getting the right result immediately. The date prefix does three things: auto-sorts, enables date-range search, prevents conflicts.

## Key Ideas
- Format: YYYY-MM-DD-[TYPE]-[TOPIC].md
- Examples: 2026-05-20-daily-wednesday.md, 2026-05-18-project-website-launch.md
- Date prefix: auto-sorts chronologically, enables date search, prevents conflicts
- Type identifier: daily, meeting, project, area, resource, book, course, idea, task
- Combined with folder structure and properties: covers almost every retrieval scenario

## Tradeoffs
- Longer filenames
- Requires discipline to apply consistently
- Retroactive renaming of existing notes is tedious

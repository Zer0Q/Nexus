# Cyril XBT Obsidian Vellum Integration

## Summary
Guide to building an AI chief of staff using Obsidian + Vellum. Five-folder structure organized by type (articles, ideas, patterns, questions, numbers). Four core workflows: process inbox, daily briefing, weekly synthesis, connection discovery. VELLUM.md as context file. Three essential plugins: Templater, Dataview, Obsidian Git. Automated capture via Readwise, Telegram bot, Whisper.

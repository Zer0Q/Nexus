# Atenov Self-Running Obsidian Brain

## Summary
The entire secret: AI agent and notes share the same folder. Agent reads, updates, organizes notes and finds connections without manual effort. Key components: shared directory, integrated terminal in Obsidian, AGENTS.md for persistent agent context, Obsidian graph for connection discovery, Kanban with strict card templates. Three mistakes: no AGENTS.md, non-markdown formats, agent and notes in different folders.

## Key Insights
- Tab switching between agent and notes is "death by a thousand cuts"
- AGENTS.md gives the agent persistent context across sessions
- Obsidian's linked notes become a navigable graph for the agent
- Kanban cards need strict templates or the agent improvises inconsistently
- The paradigm shift: from opening apps to talking to a terminal
- Three critical mistakes: no context file, mixed formats, separate directories

# CAF Objective C - Detection

## Summary
Objetivo C del CAF: capacidades para asegurar que las defensas de seguridad siguen siendo efectivas y para detectar eventos de ciberseguridad que afecten funciones esenciales.

## Core Concepts
- [[Cyber-Assessment-Framework]] -- marco con objetivo de detección de eventos
- [[Cyber-Event-Detection]] -- capacidades para detectar eventos de seguridad cibernética

## Key Insights
- La detección es un objetivo separado de la protección, enfatizando la importancia del monitoreo continuo

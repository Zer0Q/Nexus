# Skill Curation for SelfEvolving Agents explained clearly

## Summary
Google's latest paper introduces SkillOS, a framework designed to help LLM agents evolve by learning to manage their own "memories" in the form of reusable skills. > i.e. automatically go Experiences -> Memories -> Skills SkillOS treats skill management like an operating system (OS).

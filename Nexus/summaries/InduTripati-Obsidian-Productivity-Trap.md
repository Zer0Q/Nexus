# Obsidian Productivity Trap

## Summary
Beginners fall into optimizing their vault setup instead of using it — productiveness theater through folder structures, plugins, and templates. The real value of a vault comes from connections between ideas, not collection of information. Experienced users simplify over time; the best vault is the one you actually use consistently.

## Core Concepts
- [[concepts/Productivity-Theater]] -- Endless optimization of systems (folders, plugins, templates) that masquerades as productivity but is actually procrastination
- [[concepts/Capture-Use-Gap]] -- Collecting information without connecting it; hoarding ≠ building intelligence

## Key Insights
- Beginners start with complexity; experienced users converge on simplicity
- A vault should extend thinking, not replace it or become a graveyard
- Structure should emerge from actual behavior, not be designed in advance
- The best beginner setup: one vault, fast capture, simple notes, low friction, consistent usage
- Connections compound thinking; folders do not

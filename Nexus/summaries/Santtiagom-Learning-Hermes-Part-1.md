# Learning Hermes: Core Concepts

## Summary
@santtiagom_ presenta los 6 conceptos fundamentales de Hermes Agent para empezar desde cero: SOUL.md (personalidad y principios), Memoria Persistente, Recall, Skills, Learning Loop y Profiles.

## Core Concepts
- [[tools/Hermes-Agent]] -- Framework de agente AI con memoria persistente, learning loop y multi-profile
- [[concepts/SOUL-MD]] -- Archivo que define la personalidad, principios y forma de trabajar del agente
- [[concepts/Memory-Persistence]] -- Guardar información importante sobre el usuario y proyecto entre sesiones
- [[concepts/Recall]] -- Capacidad de recuperar conversaciones pasadas cuando se necesita más contexto
- [[concepts/Skill-Files]] -- Formas reutilizables de resolver tareas, guardadas como skills
- [[concepts/Learning-Loop]] -- El agente guarda aprendizajes, mejora skills existentes o crea nuevas
-  -- Agentes separados con su propia memoria, skills y configuración

## Key Insights
- SOUL.md define quién es el agente: personalidad, principios y forma de trabajar
- Memoria persistente guarda información entre sesiones — no se pierde al cerrar la sesión
- Recall permite recuperar conversaciones pasadas para contexto adicional
- Skills son instrucciones reutilizables para tareas frecuentes
- Learning Loop permite al agente mejorar con el uso: guardar aprendizajes como nuevas skills
- Profiles permiten tener múltiples agentes separados con configuraciones independientes

# Hermes Learning Loop

## Summary
@santtiagom_ explica el learning loop de Hermes: la habilidad más interesante del framework es que las Skills mejoran con el uso. Si el agente encuentra una forma útil de hacer algo o le corrigís algo, puede guardar ese aprendizaje como una nueva skill o mejorar una existente.

## Core Concepts
- [[concepts/Learning-Loop]] -- Convertir experiencia en capacidades reutilizables que el agente puede volver a usar
- [[concepts/Skill-Files]] -- Instrucciones que pueden mejorar con el uso mediante el learning loop
-  -- El agente no solo ejecuta tareas sino que aprende de la experiencia

## Key Insights
- Ejemplo práctico: al revisar PRs, si aparecen seguido problemas de permisos, validaciones o rate limits, ese aprendizaje mejora la skill de review
- El aprendizaje puede mejorar una skill existente o convertirse en algo más específico (ej: "review de endpoints públicos")
- Lo interesante de Hermes vs otras herramientas: la skill puede mejorar con el uso, no es estática
- El punto del learning loop: convertir experiencia en capacidades que el agente puede volver a usar

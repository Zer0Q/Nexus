# Omar Saridakis Research KB

## Summary
Omar Saridakis (DAIR.AI) describes his automated research paper curation pipeline: a fine-tuned Skill captures high-signal papers daily, indexed via qmd CLI tool for semantic search, then leveraged inside an interactive artifact generator with MCP tools. The knowledge layer feeds autoresearch loops, making research questions better through accumulated insights.

## Core Concepts
- [[concepts/Automated-Research-Curation]] -- AI-curated paper discovery replacing manual review
- [[concepts/Write-Time-Knowledge-Systems]] -- papers indexed once, queried many times
- [[Knowledge Layer For Agents]] -- KB as input to agentic research loops
- [[Visual Knowledge Artifacts]] -- interactive visualizations over KB data

## Key Insights
- Months of Skill tuning produce a curation agent that captures "best of the best"
- qmd CLI provides hybrid BM25/vector search over markdown KB
- Research quality depends on research questions, which depend on KB insights
- Interactive visual artifacts allow dynamic exploration of 100s of papers
- The KB layer amplifies autoresearch by providing better starting questions

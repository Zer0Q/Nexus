# NCSC CAF Objective D — Minimising Impact

## Summary
One of the 5 core objectives of the NCSC Cyber Assessment Framework. Objective D requires capabilities to minimise the adverse impact of a cyber security incident on essential functions, including restoration of those functions where necessary. The source page is a framework overview — detailed criteria are in sub-principles not captured here.

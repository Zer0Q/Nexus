# NCSC CAF Objective C — Detecting Events

## Summary
One of the 5 core objectives of the NCSC Cyber Assessment Framework. Objective C requires capabilities to ensure security defences remain effective and to detect cyber security events affecting, or with the potential to affect, essential functions. The source page is a framework overview — detailed detection criteria are in sub-principles not captured here.

## Key Insights
- Objective C covers two dimensions: maintaining defence effectiveness AND detecting events — prevention and detection are paired
- "Potential to affect" is key — detection must cover precursors and indicators, not just confirmed incidents
- This is one of 5 CAF objectives (A-E), each covering a different security capability area

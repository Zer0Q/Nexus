# Obsidian Skills Plugin

## Summary
Obsidian Skills plugin by @kepano provides a framework for creating persistent, task-oriented AI agents inside Obsidian vaults. Skills are specialized agents with access to specific notes/folders, programmed via natural language instructions to perform tasks like summarization, research analysis, content generation, or vault organization. Uses Obsidian's plugin architecture and metadata for deep knowledge graph awareness.

## Core Concepts
- [[concepts/Skill-Based-AI-Agents]] -- specialized agents with scoped vault access
- [[concepts/Persistent-AI-Helpers]] -- saved instructions and target data for reuse
- [[concepts/Vault-Aware-Automation]] -- agents that understand the knowledge graph
- [[concepts/Markdown-as-Universal-Format]] -- .md as the bridge format

## Key Insights
- Skills are not one-off prompts -- they are persistent, reusable agents
- Four use cases: auto-summarizer, research assistant, content generator, vault organizer
- Skills leverage Obsidian's existing plugin architecture and metadata
- Requires OpenAI API key or compatible LLM provider
- Barrier to entry: need Obsidian familiarity + API keys, but payoff is personalized automation

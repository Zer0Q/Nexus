# NCSC CAF Terms and Definitions

## Summary
Glossary of key terms used within the NCSC Cyber Assessment Framework (CAF) Version 4.0 (document references v3.2 content). Terms are organized alphabetically, with definitions specific to CAF context — some terms have different meanings outside the framework. The NCSC Glossary is the primary source for common cybersecurity definitions; this document covers CAF-specific usage only. Definitions are periodically updated with each CAF version release.

## Key Insights
- CAF-specific definitions may differ from common cybersecurity usage — always check CAF glossary first
- The NCSC Glossary is the primary source; CAF glossary only covers terms where CAF usage differs from standard definitions
- Document is maintained by the NCSC Support to Regulation Team (StRT@ncsc.gov.uk)
- Where external sources (NIST, ISO) don't have appropriate definitions, NCSC creates CAF-tailored definitions
- Alphabetical organization, not thematic — makes it hard to find related concepts without the full text

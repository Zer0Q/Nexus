# Industrial Cybersecurity Standards Overview

## Summary
Comparativa de estándares internacionales de ciberseguridad para sistemas industriales: ISA/IEC 62443 como estándar principal para automatización y control industrial (IACS), y NIST SP 800-82 como guía complementaria del NIST para proteger infraestructuras críticas con tecnología OT. Ambos se alinean con ISO 27001/27002 adaptados a entornos industriales, y son referenciados por el NCSC CAF como estándares complementarios para evaluaciones de resiliencia cibernética.

## Core Concepts

- [[concepts/ISA-IEC-62443]] -- Estándar internacional para ciberseguridad en sistemas de automatización y control industrial (IACS)
- [[concepts/NIST-SP-800-82]] -- Guía NIST para proteger infraestructuras críticas con tecnología de control industrial
- [[concepts/ISO-27001]] -- Estándar base de gestión de seguridad de la información, adaptado a entornos industriales
- [[concepts/Cyber-Security-Governance]] -- Gobernanza de ciberseguridad aplicada a infraestructuras industriales críticas
- [[concepts/Regional-Cyber-Regulations]] -- Regulaciones regionales que hacen referencia a estos estándares industriales

## Key Insights

- ISA/IEC 62443 cubre gestión de seguridad, arquitectura de sistemas, componentes y conexiones de red en entornos IACS
- NIST SP 800-82 (Revisión 2) actualiza las recomendaciones para entornos ICS/OT modernos
- Los dos estándares son complementarios, no competidores: ISA/IEC 62443 es más prescriptivo, NIST SP 800-82 es más orientado a guías
- Ambos se construyen sobre ISO 27001/27002 adaptándolos a entornos industriales específicos
- El NCSC CAF los referencia como estándares complementarios para evaluaciones de resiliencia cibernética
- La convergencia IT/OT exige que los equipos de seguridad entiendan ambos mundos

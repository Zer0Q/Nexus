# Structured Source Notes

## Definition
A consistent template applied to every processed article, paper, or transcript, containing fields like core argument, key points, connections to existing notes, and open questions. The structure is designed to be machine-readable so AI can reason across sources.

## Why It Matters
Inconsistent formatting breaks automation. When every source follows the same structure, AI can scan hundreds of notes and reliably extract connections, contradictions, and patterns. Without structure, the AI cannot distinguish key claims from commentary.

## Key Ideas
- Template fields: title, source, date, type, core argument, key points, my reaction, connections, open question
- The connections field is the most important -- it links new sources to existing knowledge
- Consistent structure enables [[frameworks/Weekly-AI-Synthesis]] and [[concepts/Context-Aware-AI-Research]]
- Processing a source into the template is the work that replaces manual note-taking
- Leave personal reaction fields blank for human input

## Tradeoffs
- Template processing takes time per source (5-15 minutes)
- Over-templating can lose nuance that does not fit the fields
- Requires discipline to apply template consistently

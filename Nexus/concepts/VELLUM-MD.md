# VELLUM.md

Context file for Vellum AI, equivalent to CLAUDE.md. Teaches the AI about vault structure, user identity, goals, and working preferences.

- Lives in 04-VELLUM/ directory
- Sections: identity, vault structure, current projects, what you want from AI
- Quality of VELLUM.md directly determines AI output quality
- Must be updated regularly -- stale context produces stale output

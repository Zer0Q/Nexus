# Kimi K2.6

An open-source, native multimodal agentic model from Moonshot AI. Advances practical capabilities in long-horizon coding, coding-driven design, proactive autonomous execution, and swarm-based task orchestration. Accessible via DeepInfra API with a 256K context window.

- 300 concurrent sub-agents, 4000-step horizon (triple K2.5's ceiling)
- Demonstrated traces: 5-day continuous-ops agent, 12-hour Zig port, 13-hour exchange-core refactor
- Capable of visual-to-code generation from sketches and screenshots
- Free, open-source, API-accessible

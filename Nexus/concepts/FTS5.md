# FTS5

Full-Text Search version 5, a SQLite extension that enables fast full-text search over large text corpora. Used by Hermes Agent as Tier 2 memory to index and search across all conversation history — every prompt, response, and tool call is searchable on demand.

- Part of the [[concepts/Agent-Multi-Tier-Memory]] system (Tier 2)
- Pairs with LLM summarization for context-aware results
- Unlimited capacity — every session is indexed

See also: [[concepts/Session-Recall]], [[concepts/Agent-Multi-Tier-Memory]]

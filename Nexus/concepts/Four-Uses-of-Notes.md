# Four Uses of Notes

## Definition
A framework defining four concrete outcomes that demonstrate a note has been used: decision support, writing fuel, conversation material, and action trigger.

## Why It Matters
Most note systems measure success by note count or organizational neatness. This framework replaces vague "usefulness" with four testable categories. A note that contributes to none of these four outcomes should not be in the vault.

## Key Ideas
- **Decision support**: Note surfaces when making a decision, providing relevant context, evidence, or perspective
- **Writing fuel**: Note contributes content, evidence, structure, or insight to something being written
- **Conversation material**: Note gives you something specific to say in a meeting, presentation, or discussion
- **Action trigger**: Note surfaces an idea at the moment when acting on it is possible

## Tradeoffs
- Demanding standard: most notes in existing vaults fail this test
- May push toward over-processing if applied too rigidly to reference material

# Local LLM

A large language model running on your own hardware rather than a cloud provider's servers. Enabled by open-source models (Llama, Mistral, DeepSeek) and local runners (LM Studio, Ollama).

- Zero data leaves your machine
- Generally less capable than frontier cloud models
- Requires 16GB+ RAM for comfortable operation
- Exposes OpenAI-compatible API at localhost

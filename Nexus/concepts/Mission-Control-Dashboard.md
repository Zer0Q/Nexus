# Mission Control Dashboard

## Definition
Custom dashboards built by Hermes Agent on demand. Describe what you want in natural language — content pipeline, memory wiki, docs page — and the agent constructs a working dashboard interface with no code required.

## Why It Matters
Eliminates the need to build custom tooling manually. Instead of spending evenings coding a dashboard, you describe what you need and Hermes constructs it. The dashboard evolves as your needs change.

## Key Ideas
- Natural language description → working dashboard
- Example: "Build a mission control with content pipeline, memory wiki, and docs page"
- Add tools as you need them — dashboard grows organically
- Becomes a custom interface built for how you work
- No code required — describe, it builds
- Integrates with existing Hermes skills and memory

## Tradeoffs
- Dashboard quality depends on description clarity
- Complex dashboards may need iterative refinement
- Custom dashboards aren't portable between Hermes instances

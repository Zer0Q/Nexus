# Code Overload

## Definition
Fenómeno donde los equipos de desarrollo producen tanto código tan rápidamente con IA que se vuelve ingobernable: más bugs, más incidentes, más deuda técnica y más code churn (código escrito y rápidamente revisado o eliminado).

## Why It Matters
Más código generado por IA no equivale a más productividad. Sin verificación y gobernanza adecuadas, la aceleración de IA amplifica problemas existentes.

## Key Ideas
- Acuñado por Mike Isaac y Erin Griffith (NYT, abril 2026)
- Equipos que reconstruyeron su trabajo alrededor de agentes IA se están ahogando en code churn y agujeros de seguridad
- El aumento de código sin verificación apropiada genera más trabajo de corrección que el ahorro inicial
- La solución no es menos IA, sino mejor orquestación y verificación

# Active Recall

## Definition
A learning technique where the learner deliberately retrieves information from memory rather than passively reviewing it. Coined in learning science research, notably by Roediger and Karpicke (2006), who showed that testing oneself produces significantly better retention than re-studying.

## Key Properties
- The act of retrieval strengthens memory more than re-exposure
- Questions requiring application test understanding; questions requiring definition test only familiarity
- Self-generated questions tend to be too easy; externally generated questions (e.g., by Claude) produce harder, more effective tests
- The discomfort of failing to recall is evidence that learning is occurring

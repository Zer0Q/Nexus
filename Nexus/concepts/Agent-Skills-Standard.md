# Agent Skills Standard

Estándar cross-platform para skills de agentes IA. Skills son archivos markdown con frontmatter (nombre + descripción) y cuerpo de instrucciones. Mismo formato funciona en Codex, Claude Code, Gemini CLI y Cursor.

- Lanzado para Codex en diciembre 2025, ahora estándar cross-platform
- Dos niveles: global (~/.agents/skills/) y project-level (dentro del proyecto)
- La descripción es lo más importante: Codex matchea tareas contra la descripción para invocación implícita

# Cloud Security

Confianza en las medidas de ciberseguridad implementadas por proveedores de servicios en la nube. Incluye evaluación de assurance schemes (CSA STAR), investigación académica e información del proveedor.

- Principio 8 de seguridad cloud del NCSC: supply chain
- Control nuevo en ISO 27001:2022 (A.5.23): Information Security for Use of Cloud Services

See also: [[Supply-Chain-Security]], [[ISO-27001]]

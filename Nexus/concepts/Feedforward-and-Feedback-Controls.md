# Feedforward and Feedback Controls

## Definition
Feedforward controls guide an agent before it acts; feedback controls observe the result after it acts and help it self-correct.

## Why It Matters
An agent with only feedback repeats preventable mistakes, while an agent with only feedforward rules never learns whether those rules worked. Reliable harnesses need both prevention and detection.

## Key Ideas
- Guides include AGENTS.md, skills, reference docs, how-tos, and deterministic scripts.
- Sensors include tests, linters, static analysis, logs, browser checks, and review agents.
- Computational controls are fast and deterministic.
- Inferential controls add semantic judgment but cost more and vary more.

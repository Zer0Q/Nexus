# x402 Payments

Pay-per-use payment protocol for AI agents. Set up an agentic wallet with one command, seed with USDC, and access hundreds of premium tools at cents per call without subscriptions. Solves the tool access economics problem for autonomous agents.

- Agentic wallet with one-install setup
- Seed with $5-10 USDC
- Access premium tools (Nansen, Exa, Firecrawl, Surf, BlockRun)
- Cents per call vs monthly subscriptions
- Tools aggregator eliminates tool discovery friction

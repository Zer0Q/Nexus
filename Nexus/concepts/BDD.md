# BDD

Behavior-Driven Development. Methodology using executable behavior specifications (e.g., Cucumber/Gherkin) to align product, QA, and development teams. AI can generate and validate BDD scenarios.

- Executable behavior specs
- Business rule validation
- AI can generate scenarios (risk: plausible but false scenarios)

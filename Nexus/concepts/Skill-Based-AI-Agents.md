# Skill-Based AI Agents

## Definition
AI agents that operate through a system of reusable skills — modular, composable procedures that define specific capabilities. Each skill encapsulates a workflow (e.g., processing articles, reviewing code) that the agent can invoke contextually.

## Why It Matters
Transforms AI from a general-purpose chatbot into a specialized assistant with repeatable, auditable workflows. Skills are versionable, shareable, and can be combined for complex multi-step tasks.

## Key Ideas
- Skills are markdown files defining triggers, steps, and templates
- Composable: complex workflows chain multiple skills together
- Observable: each skill's output is traceable and reviewable

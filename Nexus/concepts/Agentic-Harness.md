# Agentic Harness

The complete scaffolding around an LLM model that enables autonomous task execution — including prompts, tools, context policies, hooks, sandboxes, subagents, and recovery paths.

- Model is just one input; harness is the rest
- Good harness + decent model > great model + bad harness
- Includes: system prompt, tool definitions, memory management, session control
- Open source reference: agentic-harness on GitHub

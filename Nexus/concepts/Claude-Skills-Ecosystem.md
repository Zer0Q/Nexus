# Claude Skills Ecosystem

## Definition
The growing ecosystem of reusable skill definitions for Claude — from official Anthropic skills (doc-coauthoring, frontend-design) to community-contributed skills on GitHub.

## Why It Matters
Skills transform Claude from a chatbot into an extensible operating system. Knowing which skills exist and how to chain them separates casual users from power users.

## Key Ideas
- Official Anthropic skills: doc-coauthoring, skill-creator, frontend-design, docx
- Community ecosystem: hundreds of skills on GitHub
- Skill security auditor: validates skills before installation
- Self-improving agent skills: agents that update their own skills
- Skills are Markdown files with YAML frontmatter

## Tradeoffs
- Quality varies across community skills
- Security risks from unvetted skills
- Skill conflicts when multiple skills target same trigger

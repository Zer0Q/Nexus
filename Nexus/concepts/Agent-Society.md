# Agent Society

## Definition
An Agent Society is a set of simulated characters or actors modeled as agents with goals, relationships, memories, secrets, and decision rules inside a shared environment.

## Why It Matters
Static NPCs follow scripts; agent societies can generate interactions from motivations and constraints. In world-building and simulation, this makes a setting explorable and stress-testable rather than merely documented.

## Key Ideas
- Each major actor needs long-term goals, short-term objectives, and private knowledge.
- Relationships and faction rules define interaction protocols.
- The society should respect the [[concepts/World-Bible]] so emergent behavior does not break canon.
- Useful for interactive fiction, game design, tabletop settings, and scenario simulation.

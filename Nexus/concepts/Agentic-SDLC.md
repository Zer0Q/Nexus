# Agentic SDLC

## Definition
Software Development Life Cycle where AI agents participate in analysis, coding, testing, and PR creation. Agents operate within a mature CI/CD pipeline, handling well-scoped tasks from issue to pull request.

## Why It Matters
Represents the evolution from AI-assisted coding (single prompt) to AI-driven development (end-to-end agent workflows). Requires governance, security, and evaluation infrastructure to operate safely.

## Key Ideas
- Agents handle full task lifecycle: issue -> plan -> code -> tests -> PR
- Requires mature CI/CD for quality gates and automated evaluation
- Best for well-scoped tasks in existing codebases
- Governance and security are the primary challenges

## Tradeoffs
- Quality varies based on task scoping and test coverage
- Requires sandbox environments for safety
- Governance overhead for enterprise adoption

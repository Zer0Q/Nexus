# Connection Cost

The cumulative time and mental effort spent routing information between disconnected tools. Scales with the square of the number of tools.

- Invisible during setup, compounds over time
- Includes context switching, manual linking, duplicate searching
- Eliminated by [[tools/Unified-Knowledge-Environment]]

See also: [[concepts/Connection-Cost]], [[tools/Unified-Knowledge-Environment]]

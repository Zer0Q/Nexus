# Local AI Mindset Shift

## Definition
The behavioral change that occurs when you move from cloud-based AI (where every API call costs money) to owned local hardware (where the marginal cost per inference is essentially electricity). Cloud pricing teaches you to ration; owning the box removes that hesitation — and that's usually where the actual money hides.

## Why It Matters
The real value of local AI isn't just the dollar savings on paper — it's the unlocked usage patterns. When a 70B model costs you nothing per call, you stop thinking twice before letting an agent loop, before re-running the whole archive, before tuning on a hunch.

## Key Ideas
- Cloud pricing teaches rationing: you think twice before every API call
- Own the box and that hesitation disappears
- Unlocked patterns: agent loops, full-archive re-runs, experimental fine-tuning, overnight runs without cost anxiety
- For AI work sellers: unit cost is electricity, not API tokens — every customer is margin
- For sensitive data: contracts, legal review, patient records, financial books — never cross your network
- The hesitation removal is usually where the actual money was hiding

## Tradeoffs
- Requires upfront hardware investment ($2,999 for DGX Spark)
- Not everyone needs this shift — casual 7B users are fine with cloud or cheap edge devices
- Local models may be less capable than frontier cloud models for some tasks

# Codex Agent

Plataforma de codificación agéntica de OpenAI con CLI, app desktop, extensión IDE (VS Code/Cursor/Windsurf) y agente cloud. ~4M usuarios semanales (abril 2026). Competidor directo de Claude Code.

- Tres modos de thread: Local (directo), Worktree (aislado en Git), Cloud (remoto)
- Built-in browser para QA automatizado
- Automations: prompts programados con cron para flujos end-to-end
- Plan mode: el agente brainstormea y produce un plan antes de escribir código

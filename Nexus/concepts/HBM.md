# HBM (High Bandwidth Memory)

Stacked memory architecture used in datacenter GPUs (H100: 3.35 TB/s). Provides dramatically higher bandwidth than unified memory or consumer VRAM.

- Determines decode speed when the model fits
- Not available on consumer hardware
- HBM bandwidth is the key differentiator between consumer and datacenter inference

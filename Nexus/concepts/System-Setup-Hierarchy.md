# System Setup Hierarchy

## Definition
The actual path to better AI output is: better system setup → better prompts → dramatically better output. Most people skip the first step and try to optimize prompts on top of nothing.

## Why It Matters
People equate "better prompts" with "better results." The reality is that prompt quality has diminishing returns without a solid system foundation. A lazy prompt on top of a well-configured system outperforms a carefully crafted prompt in a blank session.

## Key Ideas
- **Wrong path:** better prompts → better output (optimizing on top of nothing)
- **Right path:** better system setup → better prompts → dramatically better output
- System setup (CLAUDE.md, AGENTS.md, custom instructions) is the multiplier
- Even a lazy prompt produces good output when rules are already in place
- The 15-minute setup investment has permanent compound returns

## Tradeoffs
- Upfront time cost (15 minutes per project) vs. long-term savings
- System setup is project-specific — no universal config works everywhere

# ISO 27001

## Definition
Estándar internacional para Sistemas de Gestión de Seguridad de la Información (ISMS). Define requisitos para establecer, implementar, mantener y mejorar continuamente la seguridad de la información. El Anexo A lista controles de seguridad clasificables.

## Why It Matters
Es el estándar de referencia global para gestión de seguridad de la información, usado como base para certificación y cumplimiento regulatorio.

## Key Ideas
- Evolución: 2005 (133 controles, 11 dominios) → 2013 (114 controles, 14 dominios) → 2022 (93 controles, 4 categorías)
- Categorías 2022: Organizacional (37), Personal (8), Físico (14), Tecnológico (34)
- 11 controles nuevos en 2022, 24 fusionados, 58 revisados
- Statement of Applicability (SoA) obligatorio para certificación
- Se complementa con ISO 27002 (guía de controles)
- Alineado con Annex SL (estructura de altos niveles)
- Atribución taxonómica por control para mapeo con NIST, CIS y otros estándares

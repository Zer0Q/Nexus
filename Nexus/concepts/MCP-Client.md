# MCP Client

## Definition
Aplicación AI (Claude, ChatGPT, VS Code, Cursor) que consume tools y data de MCP servers a través del Model Context Protocol.

## Why It Matters
MCP estandariza la conexión entre AI apps y sistemas externos — como un puerto USB-C para AI. Un client puede conectar a múltiples servers sin código custom por cada integración.

## Key Ideas
- AI assistants: Claude, ChatGPT
- Development tools: VS Code, Cursor, MCPJam
- Conecta a data sources (files locales, databases), tools (search, calculators) y workflows (prompts especializados)
- Build once, integrate everywhere — un server sirve a múltiples clients

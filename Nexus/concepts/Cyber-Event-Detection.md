# Cyber Event Detection

Capacidades para asegurar que las defensas de seguridad siguen siendo efectivas y para detectar eventos de ciberseguridad que afecten o puedan afectar funciones esenciales. Objetivo C del CAF.

- Diferente de protección: la detección es monitoreo continuo y respuesta temprana
- Incluye validación de efectividad de defensas existentes

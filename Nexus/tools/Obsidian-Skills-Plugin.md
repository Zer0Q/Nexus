# Obsidian Skills Plugin

## Definition
Obsidian plugin by @kepano providing a framework for creating persistent, task-oriented AI agents inside Obsidian vaults. Skills are specialized agents with scoped access to specific notes/folders, programmed via natural language instructions.

## Why It Matters
Moves beyond one-off AI prompts to persistent, reusable agents that understand vault context. Skills are saved with their instructions and target data, runnable again anytime.

## Key Ideas
- Four use cases: auto-summarizer, research assistant, content generator, vault organizer
- Skills leverage Obsidian's plugin architecture and metadata
- Requires OpenAI API key or compatible LLM provider
- Beta plugin, installable via Community Plugins or GitHub
- GitHub: github.com/kepano/obsidian-skills
- Skills are persistent -- instructions and target data saved for reuse

## Tradeoffs
- Requires API keys and Obsidian familiarity
- Beta stage -- may have stability issues
- Less flexible than custom agent setups (e.g., Hermes Agent + Obsidian)

# mcpvault

## Definition
MCP server sin dependencias que lee vaults de Obsidian sin necesidad de tener Obsidian running. Búsqueda BM25 con relevance ranking, maneja frontmatter safely, 14 métodos MCP.

## Why It Matters
Es la opción MCP más limpia para la mayoría de usuarios — conecta Claude Desktop, Claude Code, Cursor y Windsurf directamente al vault de Obsidian sin overhead.

## Key Ideas
- Zero-dependency — no necesita Obsidian running
- BM25 search con relevance ranking
- 14 MCP methods
- Maneja frontmatter safely
- Compatible con Claude Desktop, Claude Code, Cursor, Windsurf
- Parte del stack diario de varios usuarios: 6 plugins + mcpvault + 2 automatizaciones

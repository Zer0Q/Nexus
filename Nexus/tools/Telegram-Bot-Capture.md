# Telegram Bot Capture Infrastructure

## Definition
A Telegram bot that accepts forwarded links, tweets, or messages and automatically creates notes in an Obsidian vault's inbox folder within seconds. The bot serves as a cross-device, one-tap capture mechanism.

## Why It Matters
Capture friction is the #1 reason knowledge systems fail. A Telegram bot removes every possible excuse: phone, desktop, any device -- forward the link and it lands in your vault. If saving something takes more than ten seconds, it will not get saved.

## Key Ideas
- Works from any device with Telegram installed
- Handles tweets, browser links, and free-text messages
- Notes land in inbox within 30 seconds
- Built with Zapier, Make, or custom bot code
- Single most important infrastructure piece in a capture-first system

## Setup Options
- Zapier/Make: no-code, connects Telegram to Obsidian API
- Custom bot: more control, requires development
- Can include metadata: timestamp, source URL, one-line context note

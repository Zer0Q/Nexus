# Open SWE

## Definition
Open-source asynchronous coding agent that transforms GitHub issues into plans, code, tests, and PRs. Part of the agentic coding ecosystem.

## Why It Matters
Demonstrates the issue-to-PR automation pattern for open-source and internal repos. Useful for delegating well-scoped maintenance tasks.

## Key Ideas
- GitHub issue -> plan -> code -> tests -> PR
- Asynchronous operation
- Variable PR quality depending on task scope
- Best for defined, bounded tasks

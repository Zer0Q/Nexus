# LM Studio

## Definition
A free desktop application that downloads and runs open-source large language models locally on your machine, exposing an OpenAI-compatible API at `http://localhost:1234`. Supports Mac, Windows, and Linux.

## Why It Matters
LM Studio is the bridge between cloud AI and local AI. It makes running local models accessible to non-technical users through a graphical interface while providing the API compatibility needed to connect existing tools.

## Key Ideas
- Download models from built-in discovery tab (Mistral 7B, Llama 3 8B, Phi-3 Mini, DeepSeek Coder)
- Model files range from 4-8GB
- Runs local server identical in format to OpenAI API
- Start/stop server from the app
- No coding required to get running

## Hardware Requirements
- Apple Silicon (M1/M2/M3): 7B-13B models run smoothly
- 16GB RAM: 7B models run well
- 32GB RAM + GPU: 13B-34B models comfortable

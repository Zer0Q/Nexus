# Copilot Plugin

## Definition
An Obsidian plugin providing an inline AI assistant that lets you select text and ask a local or cloud model to summarise, expand, rewrite, or improve it. Supports slash commands inside notes.

## Why It Matters
Inline AI editing is faster than switching to a separate chat window. You can improve a paragraph without leaving your note.

## Key Ideas
- Select text, ask model to edit inline
- Slash commands for quick actions
- Connects to LM Studio at `http://localhost:1234/v1`
- Works with both local and cloud models

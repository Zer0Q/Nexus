# Nous Portal

A flat-rate model access service built by the Hermes team: $20/month for curated models with no API key management. Positioned as the starting point for users who want one bill and no provider juggling.

- $20/month flat rate
- Curated model selection maintained by Nous Research
- Built into Hermes Agent
- Alternative to per-token billing from Anthropic, OpenAI, etc.

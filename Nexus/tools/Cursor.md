# Cursor

A code editor rebuilt around AI agents (Cursor 3, released April 2026). Not a chatbot -- you type instructions inside the Agents Window while your codebase is open. The agent reads actual files, makes changes to actual code, and opens real pull requests.

- Agents Window: parallel agents running simultaneously across different codebase areas
- Each agent runs in its own Git worktree to prevent conflicts
- Cloud Handoff: offload long tasks to Cursor's infrastructure
- Design Mode: click-to-edit UI elements directly in the browser
- Agent users outnumber Tab autocomplete users 2:1
- Pro plan: $20/month, used by 64% of Fortune 500

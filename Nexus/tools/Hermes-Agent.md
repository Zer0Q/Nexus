# Hermes Agent

## Definition
Framework de agente AI con memoria persistente entre sesiones, learning loop (el agente mejora con el uso), recall de conversaciones pasadas, skills reutilizables y multi-profile (agentes separados con su propia configuración).

## Why It Matters
Hermes Agent implementa el learning loop de forma nativa: las skills pueden mejorar con el uso, convirtiendo experiencia en capacidades reutilizables. El learning loop es lo que diferencia a Hermes de otros frameworks.

## Key Ideas
- SOUL.md define personalidad, principios y forma de trabajar
- Memoria persistente guarda información entre sesiones
- Recall permite recuperar conversaciones pasadas
- Skills pueden mejorar con el uso (learning loop)
- Profiles permiten múltiples agentes separados
- No requiere cloud dependencies — arquitectura local-first

# BMO Chatbot Plugin

## Definition
An Obsidian plugin providing a full chat interface inside Obsidian that connects directly to a local LM Studio server. Feels like Claude or ChatGPT but runs entirely on-device.

## Why It Matters
Gives you a familiar chat experience without sending data to the cloud. Useful for drafting notes, rewriting paragraphs, generating ideas, and asking questions offline.

## Key Ideas
- Full chat interface inside Obsidian
- Connects to `http://localhost:1234/v1`
- Disable cloud fallback options for strict privacy
- Supports drafting, rewriting, summarising, ideation

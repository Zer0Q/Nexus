# CLAUDE.md Project Knowledge File

## Definition
A markdown file placed in an Obsidian vault that defines the AI's context: your work focus, audience, content pillars, research areas, and preferred output format. Loaded as project knowledge so every AI session starts with your specific context.

## Why It Matters
A vague CLAUDE.md produces vague outputs. A specific one turns a general-purpose AI into a domain-aware research partner that knows your focus areas and can flag relevant connections.

## Key Ideas
- Highest-leverage configuration file in an AI-augmented vault
- Should contain: work specifics, audience, content pillars, research focus
- Loaded as project knowledge so AI has context from session start
- Vague files = vague outputs; specific files = specific outputs
- Should be updated periodically as your focus evolves

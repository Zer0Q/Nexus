# Vellum

## Definition
AI platform (vellum.ai) that can be connected to Obsidian vaults as an intelligence layer. Alternative to Claude for vault-based AI reasoning. Prepaid credits passed through at cost with zero markup.

## Why It Matters
Provides an alternative AI intelligence layer for vault-based knowledge systems. Light daily usage runs a few dollars per month. Works with VELLUM.md context files for personalized AI calibration.

## Key Ideas
- Connected to Obsidian vault folder on local machine
- VELLUM.md: context file teaching Vellum about vault structure, user identity, goals
- Four core workflows: process inbox, daily briefing, weekly synthesis, connection discovery
- Workflows saved as markdown files in 04-VELLUM/workflows/
- Light daily usage: few dollars/month
- Credits passed through at cost, zero markup

## Tradeoffs
- Prepaid credit model vs. pay-per-use
- Less ecosystem maturity than Claude
- May lack some automation features (e.g., N8N integration)

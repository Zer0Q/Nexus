# QuickAdd

An Obsidian community plugin that provides zero-friction idea capture via keyboard shortcuts. Each shortcut opens a floating input box; the idea lands in the correct section of today's daily note automatically. No navigation, no categorization at capture time.

- General Capture (Ctrl+Shift+C), Research Signal (Ctrl+Shift+R), Content Idea (Ctrl+Shift+I), Link (Ctrl+Shift+L)
- Each workflow appends to today's daily note under the matching heading
- Claude does the categorization and connection-finding later
- Extends to any device via Telegram bot + N8N forwarding to vault inbox

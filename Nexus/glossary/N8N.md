# N8N

Open-source workflow automation tool. Connects apps and services via node-based visual editor. Self-hosted or cloud.

- Used as the "pipeline" layer in four-layer vault architecture
- Cron triggers for scheduled operations (daily brief at 6am)
- File watch triggers for event-based operations (queue processing)
- Webhook triggers for external events (Stripe payments, calendar events)
- Free to self-host, unlimited workflow runs

# x402

Payment protocol for AI agents that enables pay-per-use access to premium tools.

- Agentic wallet: set up with one command, seed with $5-10 USDC
- Each tool call costs cents; no subscriptions required
- Tools aggregator: hundreds of premium tools (Nansen, Exa, Firecrawl, Surf, BlockRun)
- Eliminates tool discovery friction — no API signups, no free tier hunting
- Integrated into due diligence and research workflows

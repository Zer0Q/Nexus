# Claude Projects

An Anthropic feature that gives Claude persistent memory across every conversation within a named project. Projects store context files, operating rules, and MCP connections so every conversation starts with full system context loaded automatically.

- Named project spaces with persistent memory
- Upload CLAUDE.md as project knowledge
- Install MCP servers for live tool access
- Project Instructions hold operating rules and context
- Turns Claude from a chat interface into a compounding system

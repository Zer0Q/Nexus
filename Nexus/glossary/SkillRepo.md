# SkillRepo

Centralized repository of reusable agent skills, queryable by the agent at runtime.

- Stores SKILL.md files with trigger conditions, steps, and pitfalls
- Skills can be official, community, or user-created
- Curator agent evaluates and updates skills based on performance
- Agents load skills on-demand rather than carrying all in context

# Metaprompt

A technique where you describe the desired result to the AI and ask it to draft the optimal prompt for obtaining it. Leverages the fact that the AI knows more about prompting than the user.

## Workflow
1. Describe the result you want
2. Ask the AI to write the optimal prompt
3. Review and adjust the proposed prompt
4. Execute

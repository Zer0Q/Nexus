# CLAUDE.md

Plaintext configuration file in a project root that Claude Code reads before every session, providing persistent instructions across tasks and agents.

- Five sections: Role, Style, Constraints, Workflow, Quality
- Replaces repeated prompting with persistent rules
- Available in Claude Code (project root) and Claude Projects (Project System Prompt on claude.ai)
- Equivalent patterns: AGENTS.md (Hermes), .cursorrules (Cursor), instructions.md (other tools)

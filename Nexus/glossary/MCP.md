# MCP (Model Context Protocol)

An open protocol that gives AI models live access to external tools, data sources, and services. MCP servers connect models to databases, APIs, file systems, and productivity tools, enabling the model to act on the world rather than just generating text.

- Standardized interface for tool access across different AI models
- Claude Code supports MCP server installation for research, data, and productivity integrations
- Examples: Exa and Tavily for research, CoinGecko and LunarCrush for data, Notion and Linear for productivity

# How Hermes Agent Memory Actually Works

## Summary
Most AI products talk about memory like it is one feature, Hermes Agent does something much more useful. Instead of treating memory as one vague bucket, Hermes separates always-on memory, user profile, searchable session history, and optional external memory providers into distinct layers. Because in Hermes, memory is not just about “remembering things.”

# BMAD Getting Started

## Summary
BMAD Method is an AI-powered development framework using specialized agents (Analyst, PM, Architect, Developer, QA) to guide projects through four phases: Analysis, Planning, Solutioning, Implementation. Offers three planning tracks (Quick Flow, BMAD Method, Enterprise) based on project complexity.

## Core Concepts
- BMAD provides a structured alternative to vibe coding, using PRDs, architecture docs, and story-based implementation
- BMad-Help acts as an intelligent guide that inspects project state and recommends next steps
- Four-phase workflow: Analysis (optional) -> Planning (required) -> Solutioning -> Implementation
- Three planning tracks scale from simple bug fixes (Quick Flow) to enterprise compliance systems

## Key Insights
- BMAD simulates an Agile team with AI agents, each with specialized roles
- Fresh chats per workflow prevents context contamination
- BMad-Help runs automatically at end of every workflow for next-step guidance
- Installation via npx creates _bmad/ (config) and _bmad-output/ (artifacts) directories
